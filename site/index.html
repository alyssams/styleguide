Layout

Responsive Design

Lexus.com was designed to work optimally on all screen sizes. There are four main breaking points for the site: 1204, 960, 720 and 280. Within these breakpoints there are thoughtful changes to keep the site’s look and feel consistent.

The homepage above is a great example of how our responsive layouts keep everything looking sharp while also providing the best user experience.

Destop Content Grid

All pages on Lexus.com lie on a 1032px grid with 1247px columns with 36px gutters. The grid should be center- aligned with equal spacing on the left and right

Some scenarios require a layout that does not fall exactly on the grid. While exceptions can be made, designers and developers should strive to limit how many times this happens.

Tablet Content Grid

Lexus.com in portrait view on a tablet is also set on a 12-column grid. Each column width is 38 pixels with margin sizes of 24 pixels. Fixed margins of 24 pixels are assigned to the outside of the grid.

Mobile Content Grid

Lexus.com on mobile is set on a 6-column grid with 30-pixel column width and 20-pixel gutter width.
