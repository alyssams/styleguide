Typography

All the text on lexus.com is set in either Nobel, Pakt or Palatino.

Nobel is used primarily in upper case for section headers and set in Bold, Book and Regular, but it’s also occasionally used in Book Italics. For example: Testimonials and Quotes are set in Book Italics using sentence case.

Pakt is used primarily for Display text and numbers in Bold and Semibold, while Palatino is mostly used in sentence case for body copy and intro paragraphs.

If there is any confusion about what type to use, base your choice on the PSD page.
